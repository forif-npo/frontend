# FORIF Design System

**FORIF Design System**은 한양대학교 프로그래밍 중앙 동아리 FORIF의 공식 디자인 시스템입니다. 이 시스템은 대한민국 정부 공식 디자인 시스템(KRDS)을 기반으로 개발되었으며, Tailwind CSS와 TypeScript를 활용하여 직관적이고 효율적인 UI/UX를 제공합니다. FORIF Design System은 여러 프로젝트에서 활용되며, 일관된 디자인과 빠른 개발 환경을 지원합니다.

### 활용 프로젝트:

- **FORIF WEB**: 동아리의 공식 웹사이트
- **FORIF TECH-BLOG**: 기술 블로그 플랫폼
- **FORIF ADMIN**: 내부 관리 시스템

## 주요 특징

FORIF Design System은 아래와 같은 주요 특징을 갖추고 있습니다:

### 1. **정부 가이드라인 준수**

- 대한민국 정부 디지털 서비스 UI/UX 가이드라인을 엄격히 준수합니다.  
  이를 통해 공공 서비스와의 디자인 일관성을 유지하며, 사용자들에게 익숙한 경험을 제공합니다.  
  공공기관에서 사용되는 서비스와의 통합 또는 확장에 적합한 디자인을 제공합니다.

### 2. **Tailwind CSS 활용**

- **Tailwind CSS**는 클래스 기반의 유틸리티 스타일링 프레임워크로, 빠르고 유연한 디자인 구현을 가능하게 합니다.  
  이를 통해 디자인 시스템의 확장성과 커스터마이징 능력을 극대화하며, 팀 내에서 일관된 스타일을 유지할 수 있습니다.  
  추가적으로, 코드의 가독성을 높이고 CSS 중복을 줄여 개발 속도를 향상시킵니다.

### 3. **TypeScript 지원**

- **TypeScript**를 활용하여 타입 안정성을 제공합니다.  
  이를 통해 개발자는 런타임 오류를 줄이고, 대규모 애플리케이션에서도 안정적으로 코드를 작성할 수 있습니다.  
  특히 복잡한 상태 관리와 데이터 흐름이 필요한 프로젝트에서 코드 품질을 높일 수 있습니다.

## 컴포넌트

FORIF Design System은 다양한 기능을 제공하는 UI 컴포넌트를 지원합니다. 각 컴포넌트는 **Client Components**와 **Server Components**로 구분되며, 사용자 경험에 최적화된 구성 요소를 제공합니다.  
모든 컴포넌트는 **Storybook**에서 쉽게 확인하고 테스트할 수 있습니다.

---

### 1. **Client Components**

Client Components는 클라이언트 측에서만 작동하며, 사용자 인터페이스의 동적인 요소를 구현하는 데 사용됩니다.

| **컴포넌트 이름** | **설명**                                   | **사용 사례**                    |
| ----------------- | ------------------------------------------ | -------------------------------- |
| Accordion         | 여러 섹션을 접고 펼 수 있는 컴포넌트       | FAQ 페이지, 문서 구조화          |
| Button            | 다양한 스타일의 버튼 컴포넌트              | 폼 제출, 사용자 액션 요청        |
| Calendar          | 날짜 선택을 위한 달력 컴포넌트             | 예약 시스템, 일정 관리           |
| CriticalAlert     | 중요한 메시지를 표시하는 컴포넌트          | 에러 메시지, 긴급 알림           |
| Disclosure        | 콘텐츠를 접고 펼칠 수 있는 컴포넌트        | 정보 숨김/노출, 세부사항 보기    |
| FileUpload        | 파일 업로드를 위한 컴포넌트                | 이미지 업로드, 문서 제출         |
| Modal             | 팝업 모달 창을 표시하는 컴포넌트           | 사용자 확인 요청, 상세 정보 표시 |
| Pagination        | 페이지네이션을 표시하는 컴포넌트           | 데이터 테이블, 블로그 목록       |
| RadioButtonGroup  | 여러 선택지 중 하나를 선택하는 라디오 버튼 | 설문조사, 필터 옵션              |
| Select            | 드롭다운 메뉴를 선택하는 컴포넌트          | 옵션 선택, 카테고리 필터링       |
| Tabs              | 탭 형태의 콘텐츠 전환 컴포넌트             | 대시보드, 프로필 페이지          |
| TextArea          | 다중 줄 텍스트 입력을 위한 컴포넌트        | 사용자 피드백, 댓글 작성         |

---

### 2. **Server Components**

Server Components는 서버 측에서 렌더링되며, 초기 페이지 로드 시 성능 최적화와 SEO를 고려한 컴포넌트입니다.

| **컴포넌트 이름** | **설명**                               | **사용 사례**                       |
| ----------------- | -------------------------------------- | ----------------------------------- |
| colors            | 색상 관련 컴포넌트                     | 테마 설정, 스타일 가이드            |
| Body              | 주요 콘텐츠를 담는 컴포넌트            | 페이지 레이아웃, 기본 구조          |
| Detail            | 상세 정보를 표시하는 컴포넌트          | 상품 설명, 사용자 프로필            |
| Display           | 텍스트나 이미지를 표시하는 컴포넌트    | 배너, 소개 텍스트                   |
| Heading           | 제목을 표시하는 컴포넌트               | 섹션 타이틀, 페이지 제목            |
| Label             | 텍스트 라벨을 표시하는 컴포넌트        | 입력 필드 라벨, 상태 표시           |
| Link              | 링크를 나타내는 컴포넌트               | 내비게이션, 외부 링크               |
| Title             | 페이지 제목을 표시하는 컴포넌트        | 페이지 헤더, 섹션 제목              |
| Badge             | 아이콘이나 라벨을 표시하는 컴포넌트    | 알림 표시, 상태 아이콘              |
| Breadcrumb        | 현재 페이지의 경로를 표시하는 컴포넌트 | 네비게이션 트레일, 페이지 맵        |
| Checkbox          | 선택 여부를 나타내는 체크박스          | 설정 옵션, 다중 선택 필터           |
| Chip              | 작은 정보 조각을 표시하는 컴포넌트     | 태그, 키워드 표시                   |
| LinkButton        | 링크와 버튼의 기능을 결합한 컴포넌트   | CTA(Call to Action), 외부 링크 버튼 |
| Spinner           | 로딩 상태를 표시하는 스피너 컴포넌트   | 데이터 로드 중, API 호출 대기       |
| Switch            | 스위치를 이용한 토글 컴포넌트          | 설정 활성화/비활성화                |
| Tag               | 태그를 표시하는 컴포넌트               | 카테고리 표시, 필터링               |
| TextInput         | 단일 줄 텍스트 입력을 위한 컴포넌트    | 폼 필드, 검색 창                    |

## 시작하기

FORIF Design System을 프로젝트에 통합하는 방법은 간단합니다. 아래 설치 명령어를 사용하여 라이브러리를 설치한 후, 필요한 컴포넌트를 import하여 사용할 수 있습니다.

### 설치 명령어:

```bash
npm install @forif/design-system
```

### 사용 예시:

아래는 Button 컴포넌트를 사용하는 코드 예시입니다.

```jsx
import { Button } from "@forif/design-system";

function App() {
  return (
    <Button variant="primary" size="xsmall">
      버튼 내용
    </Button>
  );
}
```

더 자세한 사용법은 README 문서를 참고하세요.
